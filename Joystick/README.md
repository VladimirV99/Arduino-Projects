# Joystick

## Description

Reading analog data from a joystick and printing it to Serial Monitor.

## Joystick Pins

- **GND** - Ground
- **+5V** - 5V
- **VRx** - Horizontal tilt
- **VRy** - Vertical tilt
- **SW** - Is joystick pressed

## Wiring

- **GND** - Ground
- **+5V** - 5V
- **VRx** - Arduino pin A0
- **VRy** - Arduino pin A1
- **SW** - Arduino pin 2
